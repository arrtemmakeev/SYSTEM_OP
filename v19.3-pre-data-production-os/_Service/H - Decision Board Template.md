# H - Decision Board Template

## Роль файла

Шаблон управленческой доски решений. Используется в 00, 04, 38, 39, 55 и при любом спорном выборе.

## Главное правило

Решение не принимается по ощущению. Решение принимается по данным, расчётам, рискам и следующему действию.

## Обязательная карточка решения

```text
Decision ID:
Дата:
Вопрос:
Контекст:
Что уже известно:
Чего не хватает:
Варианты:
1. Осторожный:
2. Реалистичный:
3. Оптимистичный:
Расчёты:
Риски:
Блокеры:
Решение:
Decision Gate:
Владелец:
Срок:
Следующая проверка:
```

## Варианты решения

| Вариант | Смысл | Когда выбирать |
|---|---|---|
| Осторожный | меньше риск | нет подтверждённого спроса |
| Реалистичный | баланс риска и скорости | есть базовые данные |
| Оптимистичный | быстрый рост | есть подтверждение спроса и мощности |
| Гибридный | поэтапный запуск | часть данных есть, часть требует проверки |

## Запреты

- Нельзя ставить GO без владельца и срока.
- Нельзя ставить GO без расчёта влияния на деньги.
- Нельзя выбирать оптимистичный сценарий без capacity и cash-flow.
- Нельзя считать мнение решением.
