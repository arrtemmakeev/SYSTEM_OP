# E - Action Rules Engine

## Роль файла

Фиксирует правила реакции системы на сигналы из рынка, кухни, финансов, клиентов и команды.

## Формат правила

```text
Сигнал → Действие → Ответственный → Срок → Чат → Метрика проверки → Decision Gate
```

## Базовые правила

| Сигнал | Действие | Чат | Gate |
|---|---|---:|---|
| Food cost >28% | пересборка техкарт | 09/25/27 | FIX |
| Food cost >30% | стоп масштабирования | 09/24/25 | NO-GO |
| CAC растёт | пересборка каналов | 28 | FIX |
| Repeat <20% | аудит продукта | 08/30/32 | FIX |
| Рейтинг <4.5 | service recovery | 30/51 | FIX |
| SLA провален | стоп рекламы | 14/15/28 | HOLD |
| Cash runway <1 мес. | антикризис | 26/38/55 | NO-GO |
| Скидки >60% | антискидочный режим | 27/28 | FIX |
| Жалобы на вкус | дегустация и контроль | 08/09/30 | FIX |
| Недовес | контроль порций | 18/45 | FIX |

## Правила владельца

Каждое действие должно иметь:

- владельца;
- срок;
- метрику;
- чат-исполнитель;
- Decision Gate.

Если владельца нет — задача не существует.
