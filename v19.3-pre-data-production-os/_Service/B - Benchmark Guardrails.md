# B - Benchmark Guardrails

## Роль файла

Фиксирует нормативы проекта по стадиям: старт, 90 дней, зрелая система.

| Метрика | Старт | 90 дней | Зрелость | Красная зона |
|---|---:|---:|---:|---:|
| Food cost | ≤28% | 25–28% | ~25% | >30% |
| Repeat | считать | 25–35% | 45–50% | <20% |
| Second order | 15–25% | 25–35% | 35–45% | <15% |
| Рейтинг | 4.6+ | 4.7+ | 4.8+ | <4.5 |
| Runway | 3 мес. | 2+ мес. | резерв | <1 мес. |
| Скидочные заказы | ≤40% | ≤35% | ≤30% | >60% |
| Аренда/выручка | ≤12% | ≤10% | ≤8–10% | >15% |
| Упаковка/чек | считать | ≤7–9% | оптимум | >12% |
| CAC/LTV | считать | ≤1:3 | ≤1:4 | >1:2 |

## Правила реакции

- Food cost >28% → чат 09 + 25 + 27.
- CAC растёт без repeat → чат 28 + 32 + 34.
- Рейтинг ниже 4.5 → чат 30 + 51 + 52.
- Runway меньше месяца → чат 26 + 38 + 55.
- Упаковка съедает маржу → чат 19 + 25.
