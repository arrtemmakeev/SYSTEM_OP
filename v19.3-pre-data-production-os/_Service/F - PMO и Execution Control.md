# F - PMO и Execution Control

## Роль файла

Фиксирует систему исполнения проекта: владельцы, сроки, статусы, блокеры, weekly review и decision log.

## Статусы задач

| Статус | Значение |
|---|---|
| TODO | не начато |
| IN PROGRESS | в работе |
| BLOCKED | заблокировано |
| DONE | выполнено |
| CANCELLED | снято |

## RAG

| Цвет | Значение |
|---|---|
| Green | всё по плану |
| Amber | есть риск |
| Red | срок/результат под угрозой |

## Weekly review

1. Что было обещано.
2. Что сделано.
3. Что зависло.
4. Что блокирует деньги, качество или запуск.
5. Какое решение нужно от собственника.
6. Что делаем за следующие 7 дней.

## Правило управления

Если задача не имеет владельца, срока и метрики проверки — она не считается задачей проекта.
