# v19.3 Pre-Data Production OS

Production-пакет для проекта dark kitchen: роллы / пицца / WOK / сеты / комбо / самовывоз.

## Статус

**Architecture Ready / Data Not Ready**

Пакет готов как предзапусковая операционная система. Фактический Launch Ready нельзя ставить без реальных данных: город, район, конкуренты, аренда, поставщики, меню, техкарты, ФОТ, CAC, repeat, legal/sanitary review.

## Контекст

- Формат: dark kitchen + небольшая зона самовывоза.
- Рынок: РФ.
- Food cost: цель 25%, максимум 28%.
- CAPEX: 4 млн ₽ с резервом.
- Repeat: 45–50% как зрелая цель.
- Упаковка: WOW, отзывы, repeat, LTV.
- Мотивация: 70% оклад + KPI + переменная часть.
- Корпоративный канал исключён.

## Как запускать

1. Открыть 00 - Главный чат.
2. Активировать контрольный слой: 50, 51, 52, 53, 54, 55, 56.
3. Начать цепочку: 01 → 02 → 03 → 04.
4. После каждого чата возвращать итог в 00.
5. При HOLD / FIX / NO-GO не идти дальше механически.
6. Заполнить Real Data Intake Checklist.
7. Собрать первый Decision Board.
